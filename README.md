# **Pay2Weather**

App created to track weather conditions by utilizing a subscription service and tiers for its users.


***It has capabilities to :*** 
1. Search for weather from anywhere ***( inside API capabilities )***
2. Show forecast for 2-6 days base on user input
3. Save favourite cities to users
4. Show most searched cities, etc.
